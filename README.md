# GestoVox

AI-powered presentation assistant using hand gestures and offline speech recognition.

---

## Project Overview

GestoVox enables presenters to control slides using hand gestures while generating live speech captions locally on the computer.

---

## Features

- Gesture-controlled slide navigation
- Virtual mouse
- Offline speech-to-text
- Multi-language support
- Local AI processing

---

## Technologies Used

- Python
- OpenCV
- MediaPipe
- Whisper
- PyAutoGUI
